1️⃣ Install Required Packages

After initializing the project (npm init -y), install packages.

Example commonly used packages:

Express.js → Web server

Mongoose → Database connection

dotenv → Environment variables

Nodemon → Auto restart server

Command:

npm install express mongoose dotenv

Install development package:

npm install nodemon --save-dev
2️⃣ Example package.json After Installing Packages
{
  "name": "node-project",
  "version": "1.0.0",
  "main": "server.js",
  "scripts": {
    "start": "node server.js",
    "dev": "nodemon server.js"
  },
  "dependencies": {
    "express": "^4.18.2",
    "mongoose": "^7.0.0",
    "dotenv": "^16.0.0"
  },
  "devDependencies": {
    "nodemon": "^3.0.0"
  }
}
3️⃣ Initialize Packages in Code

Example server.js:

const express = require("express");
const dotenv = require("dotenv");

dotenv.config();

const app = express();

app.use(express.json());

app.get("/", (req, res) => {
  res.send("Node.js Server Running");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});

Development mode:

npm run dev

Production mode:

npm start
