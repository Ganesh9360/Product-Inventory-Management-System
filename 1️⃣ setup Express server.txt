1️⃣ Install Express

Use npm to install Express.

npm install express
2️⃣ Create Server File

Create a file called:

server.js
3️⃣ Basic Express Server Code
const express = require("express");
const app = express();

const PORT = 3000;

app.get("/", (req, res) => {
  res.send("Welcome to Express Server");
});

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

Explanation:

express() → Creates Express application

app.get() → Handles GET request

res.send() → Sends response

app.listen() → Starts server

4️⃣ Run the Server
node server.js

Output:

Server running on port 3000

Open browser:

http://localhost:3000

You will see:

Welcome to Express Server
5️⃣ Express Server with JSON Middleware
const express = require("express");
const app = express();

app.use(express.json());

app.get("/", (req, res) => {
  res.send("Express Server Working");
});

app.post("/data", (req, res) => {
  res.json({
    message: "Data received",
    data: req.body
  });
});

app.listen(3000, () => {
  console.log("Server started on port 3000");
});
